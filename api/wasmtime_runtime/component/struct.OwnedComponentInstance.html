Struct wasmtime_runtime::component::OwnedComponentInstance

source · 
pub struct OwnedComponentInstance { /* private fields */ }
Expand description

An owned version of ComponentInstance which is akin to Box<ComponentInstance>.

This type can be dereferenced to ComponentInstance to access the underlying methods.
